Biblio

Biblio is a website that allows individuals to insert any text they would like read aloud for free. Thanks to a javascript technology called speechsynthesis. This technology allows the users to be able to insert any book, text or article into the text area and let it be read to them by the speechsynthesis. In addition, users from all over the world will be able to make use of this technology as it will be able to read in multiple different languages such as, english, spanish, chinese and more. Therefore, the text reader is customizible to ones liking; one is able to change the tune and pitch. This means that individuals are able to make the voice more feminine or masculine. That being said, this website will be updated frequently to ensure a better user expereince.
